# Tennis Score Keeper

## Requirements

Build an app that calculates the score for Tennis.

## Run Project

```Powershell
npm run dev
```

## Test Project

```Powershell
npm run test
```

## Project Setup

```Powershell
npm init -y
npm i --save-dev nodemon karma-cli karma mocha chai karma-mocha karma-chai karma-chrome-launcher
mkdir src 
mkdir test
karma init karma.conf.js
git init
git add .
git commit -m "Add Project Files"
```
